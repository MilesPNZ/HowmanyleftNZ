What is this?

How Many Left is a tool to show how rare a specific make/model of vehicle is. The database searches cars currently registered on New Zealand Roads.

You can use this website to see how rare a particular vehicle may be and see a rarity score for your search. If you have any feedback or suggestions, or if you spot any issues or errors, please reach out using the contact form on this website.

How do I get started?

Start by searching a make or model that you are interested in using the form above or for a more refined search, visit the Advanced Search page.

You can also search by keywords or specifics for example: "Cosworth".
